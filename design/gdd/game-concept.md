# Game Concept — Office Layoff Survivor (working title)

## Elevator pitch

AI와 구조조정이 몰아치는 회사에서, **현장 개발자**가 뱀파이어 서바이벌식 탑다운 전투로 **8주(2달)** 를 버티는 인디 풍자 액션.

## Core fantasy

- 말로는 통하지 않는 압박(HR·대표·기술이사)을 **생존과 화력**으로 견딘다.
- 프롤로그: *게임 개발자는 게임으로 자신을 표현한다* — 제작 동기.

## Pillars

1. **읽히는 풍자** — 실루엣·색만으로 역할 구분.
2. **빠른 루프** — 자동 공격, 주차 진행, 짧은 세션.
3. **인디 스코프** — 정적 2D 스프라이트 → 이후 애니메이션.

## Visual identity anchor

**“회의실 형광등 아래, 픽셀·플랫 2D 치비 오피스”** — 밝기는 중간, 채도는 선명, 윤곽선 2px.

## Target platform

- PC (Windows) — Godot 4.6 .NET, 데스크톱 우선.
